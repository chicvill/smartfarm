## 1. 사업 개요

### 1.1 사업 목표
```
사업명: 스마트팜 엽채류 재배 사업
재배 작물: 상추 (청상추, 적상추)
재배 면적: 500평 (1,650m²)
사업 형태: 시설원예업 (토지 임대)
목표 생산량: 연 140톤
목표 매출: 연 7억 1,120만원
```

### 1.2 사업 타당성
```
시장 환경:
✓ 국내 상추 연 소비량: 약 30만톤
✓ 신선 채소 시장 연 성장률: 5-7%
✓ 스마트팜 생산 비중 증가 추세
✓ 안정적 수요 (외식업, 가정)

경쟁 우위:
✓ 연중 안정 생산
✓ 고품질 (무농약/저농약)
✓ 신선도 관리 (수확 당일 배송)
✓ IoT 기반 품질 균일성
```

### 1.3 핵심 성공 요소
```
1. 생산성 확보
   - 작기 단축: 25일/작기
   - 연 14회 수확
   - 폐기율 2% 이하

2. 판매처 다변화
   - 직판 30% (고마진)
   - B2B 60% (안정)
   - 가공 10% (잉여)

3. 효율적 운영
   - 최소 인력 (3.5명)
   - 자동화 시스템
   - 원가 절감

4. 품질 관리
   - 스마트팜 정밀 제어
   - GAP 인증 추진
   - 브랜드 구축
```
