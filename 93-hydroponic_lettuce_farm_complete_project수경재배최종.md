## 10. FAQ 및 트러블슈팅

### 10.1 자주 묻는 질문

```
Q1: 농업 경험이 없어도 가능한가요?
A: 가능하지만 최소 6개월 이상의 교육과 시범 재배가 필수입니다.
   - 양액재배 전문 교육 40시간 이상
   - 스마트팜 운영 교육 20시간
   - 전문가 멘토링 3-6개월
   - 관련 농장 견학 및 실습

Q2: 투자 회수 기간은 정말 2.6년인가요?
A: 아니요. 본 계획서의 2.6년은 매우 낙관적입니다.
   현실적으로는 3-5년을 예상해야 합니다.
   이유:
   - 초기 생산량 불안정
   - 판로 개척 시간 필요
   - 기술 습득 기간
   - 불량률 초기 높음

Q3: 정부 지원금은 얼마나 받을 수 있나요?
A: 조건에 따라 다릅니다.
   - 스마트팜 청년창업: 최대 1.5억원 (무이자/저이자)
   - 귀농귀촌 지원: 최대 3억원 (저이자)
   - 혁신밸리 입주: 시설 50% 지원
   → 반드시 사전에 자격 요건 확인 필수

Q4: 양액재배가 토경재배보다 정말 좋나요?
A: 장단점이 있습니다.
   장점:
   ✓ 생산성 30% 높음
   ✓ 물 70% 절약
   ✓ 품질 균일
   ✓ 토양 병해 없음
   단점:
   ✗ 초기 투자 2배
   ✗ 기술 난이도 높음
   ✗ 전기 의존
   ✗ 시스템 고장 시 타격 큼

Q5: 한 사람이 운영 가능한가요?
A: 불가능합니다. 최소 3명 필요.
   - 대표 (재배 총괄, 경영)
   - 재배 담당 (정식, 수확, 양액 관리)
   - 출하/영업 담당 (포장, 배송, 영업)
   → 바쁜 시기(수확) 추가 인력 2-3명

Q6: 여름/겨울에도 생산 가능한가요?
A: 가능하지만 비용 증가합니다.
   - 여름: 냉방비 ↑ (월 +100만원)
   - 겨울: 난방비 ↑ (월 +200만원)
   - LED 보광: 연중 안정 생산 가능
   - 계절별 품종 조절 필요

Q7: 유기농 인증 가능한가요?
A: 양액재배는 유기농 불가입니다.
   - 무농약 인증: 가능
   - GAP 인증: 가능
   - 친환경 인증: 가능
   - 유기농: 불가 (토양 재배만 가능)

Q8: 수출도 가능한가요?
A: 가능하지만 쉽지 않습니다.
   - 신선 상추: 유통기한 짧아 어려움
   - 가공 상추 (세척, 포장): 가능성 있음
   - 일본, 동남아 시장 유망
   - HACCP, GAP 인증 필수
   - 물류 비용 高

Q9: 다른 작물도 재배 가능한가요?
A: NFT는 엽채류에 최적화되어 있습니다.
   재배 가능:
   ✓ 상추, 청경채, 쑥갓
   ✓ 바질, 루꼴라, 케일
   ✓ 허브류 (민트, 타임 등)
   어려움:
   ✗ 토마토, 오이 (지지대 필요)
   ✗ 딸기 (배지경 적합)
   ✗ 뿌리채소 (부적합)

Q10: 실패 확률은 얼마나 되나요?
A: 약 40-50%로 추정됩니다.
    실패 주요 원인:
    - 판로 확보 실패 (40%)
    - 기술 부족 (30%)
    - 자금 부족 (20%)
    - 기타 (10%)
    성공 요인:
    - 착수 전 계약재배 70% 확보
    - 충분한 교육 및 실습
    - 여유 있는 자금
    - 끈기와 학습 의지
```

### 10.2 트러블슈팅

```
**증상: pH가 계속 상승함**
원인: 음이온 (NO3-, H2PO4-) 과다 흡수
해결:
1. 양액 교체
2. pH 조절제 (산) 추가
3. 양액 조성 재검토
4. 센서 캘리브레이션

**증상: EC가 계속 상승함**
원인: 물 증발 > 양액 흡수
해결:
1. 순수한 물 보충
2. 양액 일부 배출 후 물 추가
3. 습도 높이기
4. 2주마다 양액 전체 교체

**증상: 뿌리가 갈변됨**
원인: 용존산소 부족 또는 병원균
해결:
1. 에어펌프 작동 확인
2. 양액 온도 낮추기 (18-22°C)
3. 과산화수소 처리 (응급)
4. 양액 교체 & 탱크 소독

**증상: 잎이 노랗게 변함**
원인: 질소 부족 또는 철 결핍
해결:
1. EC 확인 (1.2-1.6 적정)
2. pH 확인 (5.8-6.3)
3. 양액 조성 확인
4. 철 킬레이트 추가 투여

**증상: 생육이 느림**
원인: 온도, 광량, CO2 부족
해결:
1. 온도 확인 (주간 18-22°C)
2. LED 높이 조절 (40cm)
3. CO2 농도 확인 (800-1000ppm)
4. 작기 연장 (2-3일)

**증상: 잎이 시듦**
원인: 양액 공급 중단 또는 고온
해결:
1. 펌프 작동 확인
2. 배관 막힘 점검
3. 양액 온도 확인
4. 환기 (온도 낮추기)

**증상: 데이터가 수집 안 됨**
원인: WiFi 연결 끊김, 센서 고장
해결:
1. WiFi 신호 강도 확인
2. ESP32 재부팅
3. MQTT 연결 확인
4. 센서 배선 점검
5. 센서 교체

**증상: CO2가 올라가지 않음**
원인: 실린더 빔, 밸브 고장, 환기 과다
해결:
1. 실린더 압력 게이지 확인
2. 솔레노이드 밸브 작동 확인
3. 센서 정확도 확인
4. 환기 상태 확인
5. 분사 노즐 막힘 점검
```
